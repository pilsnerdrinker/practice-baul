# P3 バウル・オブ・アゴニー 練習ページ

FF14「絶妖星乱舞」P3前半の **バウル・オブ・アゴニー** を練習するためのブラウザ用ページです。  
現時点では、以下の2方式に対応しています。

- ヤーン（TLB3）式
- ヤーン（DN / 正攻法）式

---

## できること

### 1. バウルの立ち位置練習

自分のロールを選択し、各場面でどこに立つかを確認・練習できます。

対応ロール：

- MT
- ST
- H1
- H2
- D1
- D2
- D3
- D4

表示モード：

- **Hard：立ち位置ボタン**
  - マップ上の丸を選んで回答します。
  - 判定後、正解は緑、誤答は赤で表示されます。
  - 正解を表示することもできます。
- **Normal：選択肢**
  - 各場面の理解クイズ形式で確認できます。
  - 判定または正解表示を押したときに正解が表示されます。

---

## 対応している方式

## ヤーン（TLB3）式

風クリスタルと水クリスタルを基準にした固定位置式です。

大まかな流れ：

1. MT組 / ST組に分かれる
2. 1回目処理
3. インプロージョン安置
4. 2回目処理
5. 真空波前準備
6. 真空波 / TLB3後の中央集合

このページでは、TLB3式の処理位置をマップ上で確認できます。

---

## ヤーン（DN / 正攻法）式

デバフ・秒数・優先順位で処理位置が変わる方式です。

大まかな流れ：

1. MT組 / ST組に分かれる
2. 1回目の炎水処理 + 風クリ処理
3. インプロージョン安置
4. 2回目の炎水処理 + 風クリ処理
5. 真空波前準備
6. 真空波後の最終配置

DN式では、炎水デバフの有無、18秒 / 45秒、ロール優先順位によって立ち位置が変わります。

---

## 基本仕様

この練習ページでは、バウル・オブ・アゴニーを以下の仕様として扱っています。

- 炎・水・風のクリスタルが出現します。
- クリスタルはフィールドマーカーの数字方向、つまり `1 / 2 / 3 / 4` のいずれかに出ます。
- 炎と水は必ず対角に出ます。
- 風の対角は空きになります。
- 全員に混沌の風が付きます。
- 炎デバフは2人、水デバフは2人に付きます。
- 炎 / 水は、それぞれTH側から1人、DPS側から1人に付く想定で生成しています。
- 炎水の発動秒数は、片方が18秒、もう片方が45秒です。
- 風デバフには「正面受け」「背面受け」があります。

---

## 使い方

1. 自分のロールを選びます。
2. 方式を選びます。
   - ヤーン（TLB3）式
   - ヤーン（DN / 正攻法）式
3. 表示モードを選びます。
   - 初期表示は **Hard** です。
4. 「新しい問題」を押します。
5. マップ上の丸を選んで「判定」を押します。
6. 必要に応じて「正解を表示」で正解位置を確認します。

パーティーリスト上のロールをクリックしても、自分ロールを変更できます。  
途中でロールや方式、Normal / Hardを切り替えても、基本的に現在の場面を維持したまま確認できます。

---

## Hardモードの操作

- マップ上の丸をクリックして回答します。
- 「判定」を押すと、選択した位置の正誤を表示します。
- 正解は緑、誤答は赤で表示されます。
- 「正解を表示」を押すと正解位置を表示します。
- 「次へ」で次の場面へ進みます。
- 「前へ」で前の場面に戻ります。
- 「正解したらそのまま次へ」にチェックを入れると、正解時だけ自動で次へ進みます。

---

## Normalモードの操作

Normalでは、各場面ごとに理解クイズが表示されます。

- 選択肢を選んで「判定」を押すと正誤が表示されます。
- 正解は緑、誤答は赤で表示されます。
- 「正解を表示」を押すと正解だけ表示されます。
- 各ページ下部には、その場面の立ち位置選択肢も表示されます。

---

## 注意

このページは個人制作の練習用ページです。  
実際の処理方法、採用マクロ、固定・野良の攻略法と異なる場合があります。

特に以下は、攻略情報の更新やPTごとの方針によって変わる可能性があります。

- ボス誘導位置
- 風の正面受け / 背面受けの細かい向き
- インプロージョン時の向き
- DN式の細かい立ち位置
- 最終配置の優先順位

実際に使用する際は、所属PTの処理法を優先してください。
